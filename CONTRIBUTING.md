# Contributing (Beta)

ModelHaven Studio is in an experimental Beta phase.

Contributions are welcome in the form of:
- thoughtful feedback
- critical questions
- ethical or technical concerns
- edge cases and failure modes

This is not a space for promotion or product ideas.

---

## How to Contribute

- Read the FRAMEWORK and PRINCIPLES first
- Use GitHub Issues for feedback
- Be precise and respectful
- Assume good intent

---

## What This Is Not

- Not a support forum
- Not a feature request board
- Not a debate stage

Clarity and restraint are valued.
