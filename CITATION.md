# Citation Guidance  
### UCSF & ModelHaven Studio

**Version:** v1.0  
**Last updated:** January 2026  
**Author:** Martijn Bruzzese

---

## Purpose

This document explains how to cite UCSF and ModelHaven Studio materials in a way that is stable for peer review and resilient to URL changes.

If you use UCSF ideas in writing, research, design work, or public discussion, please cite the **author + title + version + date + repository**.

---

## Recommended Canonical Citation (Generic)

**Martijn Bruzzese.** (2026).  
**Universal Consent & Safety Framework (UCSF)** — v1.0.  
ModelHaven Studio repository (GitHub).

And/or:

**Martijn Bruzzese.** (2026).  
**ModelHaven Studio — Whitepaper** — v1.0.  
ModelHaven Studio repository (GitHub).

---

## Academic Formats

### APA (example)

Bruzzese, M. (2026). *Universal Consent & Safety Framework (UCSF) (v1.0).* ModelHaven Studio (GitHub repository).

Bruzzese, M. (2026). *ModelHaven Studio — Whitepaper (v1.0).* ModelHaven Studio (GitHub repository).

### MLA (example)

Bruzzese, Martijn. *Universal Consent & Safety Framework (UCSF). Version 1.0*, 2026. ModelHaven Studio, GitHub repository.

Bruzzese, Martijn. *ModelHaven Studio — Whitepaper. Version 1.0*, 2026. ModelHaven Studio, GitHub repository.

### Chicago (example)

Bruzzese, Martijn. 2026. *Universal Consent & Safety Framework (UCSF), v1.0.* ModelHaven Studio (GitHub repository).

Bruzzese, Martijn. 2026. *ModelHaven Studio — Whitepaper, v1.0.* ModelHaven Studio (GitHub repository).

---

## How to Cite Specific Documents in This Repo

When citing a specific file, include:

- Document title
- Version (if present)
- Last updated date
- File name (e.g., `USE_CASES.md`)
- Repository name

Example:

Bruzzese, M. (2026). *Use Cases & Failure Modes — UCSF (v1.0).* `USE_CASES.md`. ModelHaven Studio repository (GitHub).

---

## Citing Translations

If you cite a translation, indicate language:

Bruzzese, M. (2026). *UCSF White Paper — English translation (v1.0).* ModelHaven Studio repository (GitHub).

If the English version is a translation of the Dutch v1.0, you may also cite both.

---

## Notes on Stability

- For peer review, use **version numbers** and **dates**.
- URLs can be included, but the citation should remain meaningful without them.
- If you publish externally, consider archiving a release/tag for stable referencing.

---

*This citation guidance supports UCSF v1.0 and may evolve through peer review.*
